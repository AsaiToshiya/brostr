== Description ==

Share chess games (PGN and Yahoo) on your Wordpress blog as an interactive chess board. Once
you have the PGN (Portable Game Notation) or the Yahoo email contents available you can create
a blog post and paste the game between the ###pgn### and %%%pgn%%% tags. The result will look
something like this, http://www.chesspastebin.com/2010/08/16/default-title-21/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the pgnviewer directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Paste PGN or Yahoo syntax chess games between the ###pgn### and %%%pgn%%% tags.
4. voila, when viewing the post these are turned into graphical boards.
